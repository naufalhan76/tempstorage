Frequently Asked Questions

Find answers to common questions about Mabox

How long are files stored?

Files are automatically deleted after 24 hours from upload. This ensures your temporary shares don't clutter our servers and maintains privacy.

If you need longer storage, you can:

  • Re-upload the file when needed
  • Use a dedicated cloud storage service for permanent storage
  • Download the file before the 24-hour limit

What's the maximum file size?

The current upload limit is 100 MB per file. This allows for most documents, images, and small media files while keeping our service fast and reliable.

For larger files, consider:

  • Compressing files using ZIP or other compression tools
  • Using specialized large file sharing services
  • Splitting large files into smaller chunks

Is my data secure?

Yes! We take security seriously:

  • HTTPS encryption for all transfers
  • Automatic deletion after 24 hours
  • No account required - no personal data collected
  • Unique URLs that are hard to guess
  • Server-side security with modern protection measures

However, avoid uploading highly sensitive data like passwords, financial information, or confidential documents.

Can I share files with multiple people?

Absolutely! Once you upload a file, you get a shareable link that you can send to anyone.

Share the link via:

  • Email
  • Chat applications (WhatsApp, Telegram, etc.)
  • Social media
  • QR codes (great for mobile sharing)
  • Any other method you prefer

Recipients don't need to create an account to download your files.

What file types are supported?

All file types are supported! Mabox is file-type agnostic.

Popular file types include:

  • Documents: PDF, DOC, DOCX, TXT, PPT, XLS
  • Images: JPG, PNG, GIF, SVG, BMP
  • Videos: MP4, AVI, MOV, WMV
  • Audio: MP3, WAV, FLAC, AAC
  • Archives: ZIP, RAR, 7Z, TAR
  • Code files: JS, HTML, CSS, PY, JSON
  • And many more!

Is there an API available?

Yes! Mabox provides a RESTful API for developers.

The API allows you to:

  • Upload files programmatically
  • Get file information
  • Download files
  • List uploaded files
  • Delete files before expiry
